Sales Insights Dashboard (Excel)
Project Overview
This project focuses on analyzing sales data to identify trends, track key performance indicators (KPIs), and provide data-driven recommendations for a retail business. By consolidating data and building an interactive dashboard in Excel, I’ve transformed complex datasets into a visual story that helps stakeholders understand regional performance and product profitability.

Key Features
Interactive Dashboard: Utilizes slicers and timeline controls to filter data by region, year, and category.

KPI Tracking: Real-time calculation of Total Revenue, Total Profit, and Profit Margin.

Automated Data Processing: Leverages Power Query and Pivot Tables to ensure the dashboard updates seamlessly with new data.

Advanced Visualizations: Includes Trend Analysis (Line Charts), Regional Breakdown (Map/Bar Charts), and Product Performance (Pareto/Column Charts).

Tech Stack
Tool: Microsoft Excel (Advanced)

Features Used: Pivot Tables, Pivot Charts, Power Query (ETL), Slicers, and DAX-style Measures (Calculated Fields).

Data Source: Sales records including Order Date, Region, Product Category, Revenue, and Cost.

Project Structure
Data Cleaning: Used Power Query to remove duplicates, handle null values, and format dates for time-series analysis.

Data Modeling: Established relationships between sales tables and product metadata.

Analysis: Created calculated columns for Profit Margin and Sales Growth % using Excel formulas.

Visualization: Built a user-friendly interface to highlight the "Top 5 Products" and "Underperforming Regions."
